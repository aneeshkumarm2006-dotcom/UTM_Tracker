📋 Manual Test Steps

  1. Start the backend server: cd site/backend && node server.js
  2. Register a user and configure in the dashboard:
    • Trigger Page: /test/get-a-quote.html
    • Button ID: wpforms-submit-7209
    • Fields: name → wpforms-7209-field_1, email → wpforms-7209-field_2, phone → wpforms-7209-field_3
  3. Copy the snippet from the Snippet page and paste it into get-a-quote.html and this file
  4. Open this page with UTM params:
    http://localhost:4000/test/snippet-test.html?utm_source=google&utm_medium=cpc&utm_campaign=test-campaign
  5. Click "Get a Quote" — UTMs are now in sessionStorage and survive the navigation
  6. Fill the form and click Submit
  7. Check the dashboard — conversion should appear with correct UTMs and captured fields
